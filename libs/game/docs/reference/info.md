# Info

Keep score, run the countdown timer, and track life status.

```cards
info.setScore(0)
info.changeScoreBy(1)
info.score()
info.highScore()
info.setLife(0)
info.changeLifeBy(1)
info.life()
info.onLifeZero(function () {})
info.startCountdown(0)
info.stopCountdown()
info.onCountdownEnd(function () {})
```
